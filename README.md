### Option 2: From source

Requires [Node.js](https://nodejs.org) and [Git](https://git-scm.com).

```bash
git clone https://github.com/slapform/html-form-backend.git
cd html-form-backend
npm install
npm start
```

## ✨ Features

- Point any HTML form at one endpoint and go
- Submissions emailed to you instantly
- No backend code, no PHP, no servers
